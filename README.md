# Women Safety Emergency System – ServiceNow

## Overview

Women Safety Emergency System is a ServiceNow-based application designed to manage emergency alerts, automate response workflows, track risk levels, and monitor safety activities through interactive reports and dashboards.

This project was built using ServiceNow App Engine Studio and Flow Designer.

---

## Features

* Emergency alert management
* Risk level tracking
* Automated workflow notifications
* Emergency response task creation
* Interactive reports and dashboards
* Real-time monitoring system

---

## Technologies Used

* ServiceNow App Engine Studio
* Flow Designer
* Business Rules
* Reports & Dashboards

---

## Modules

### Emergency Alert Table

Stores emergency details such as:

* Reporter Name
* Mobile Number
* Emergency Type
* Location
* Description
* Risk Level
* Alert Status
* Emergency Time

### Flow Automation

Automatically creates emergency response tasks when:

* Risk Level = High
* Risk Level = Critical

### Reports

* Alerts by Risk Level
* Alerts by Status
* Emergency Types Summary
* Critical Emergency Alerts

### Dashboard

Women Safety Emergency Dashboard provides:

* Real-time emergency monitoring
* Risk analytics
* Status tracking
* Emergency visualization
